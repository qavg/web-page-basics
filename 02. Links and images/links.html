Anchors

This "link to images.html" is created with:

<a href="images.html">link to images.html<a>

Links Types

  • Absolute Links

    • Example input:

    <a href="https://developer.mozilla.org/en-US/docs/Web/HTML" target="_blank">Mozilla Developer Network<a>

    Output:

    Mozilla Developer Network

  • Relative Links

    • Example input:

    Relative links, like to the <a href='misc/extras.html'>extras page</a>

    Output:

    Relative links, like to the extras page

  • Root-Relative Links

    • Example input:

    <a href='/'>Home Page</a>

    Output:

    Home Page
